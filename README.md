# Andalusia — Privacy Policy (GitHub Pages)

Kebijakan privasi resmi untuk aplikasi **Andalusia** (`com.AndalusiaProject.Andalusia`).

## URL setelah deploy

https://andalusiagame.github.io/andalusia-privacypolicy/

## Deploy ke GitHub Pages

1. Push folder ini ke repo: https://github.com/andalusiagame/andalusia-privacypolicy
2. Di GitHub: **Settings → Pages**
3. **Source:** Deploy from a branch
4. **Branch:** `main` → folder `/ (root)`
5. Save — tunggu 1–5 menit, buka URL di atas

## Play Console

Paste URL di: **App content → Privacy policy**

## Data Safety (ringkas)

- Collects: email (account), app activity / game progress, diagnostics
- No payment data
- Encrypted in transit: Yes
- Privacy policy URL: URL GitHub Pages di atas
